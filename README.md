# SpaceAccents

**SpaceAccents** — плагин для Minecraft (Paper/Spigot), который позволяет добавлять разнообразные языковые акценты и стили речи в чат.  
Идеально подходит для ролевых серверов, фанов и иммерсивных миров!

---

## ⚡ Возможности

- Преобразование чата игрока в выбранный акцент.
- Поддержка **фраз**, **слов** и **слогов** для гибкого изменения текста.
- Несколько акцентов одновременно.
- Простая кастомизация через `accents.yml`.
- Возможность добавлять несколько вариантов перевода для случайного выбора.
- Оптимизировано для серверов с большим количеством игроков.

---

## 📂 Структура `accents.yml`

Пример конфигурации:

```yaml
accents:
  nian:
    display: "Nian"
    replacements:
      - from: "с"
        to: "ш"
      - from: "ч"
        to: "щ"
  dwarven:
    display: "Dwarven"
    replacements:
      - from: "пиво"
        to: "Жидкайе Злато"
      - from: "е"
        to: "э"
      - from: "блять"
        to: "Кнурла"
